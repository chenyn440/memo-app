# 备忘录应用 PRD

## 项目概述
基于 Tauri + React + SQLite 的桌面备忘录应用，支持 Markdown 编辑，深度集成 AI 辅助能力。

## 技术栈
- 前端：React + TypeScript + Zustand
- 后端：Rust + Tauri
- AI 层：LangChain + OpenAI / 智谱 AI / DeepSeek (OpenAI 兼容接口)
- 数据库：SQLite + FTS5 全文搜索
- UI：原生 CSS（精简设计）

## 已完成功能

### v0.9.0 - 会议协作体验升级 (2026-03-26)

#### 1. 会议协作与参会管理
- [x] **参会人员列表**：桌面端在会议页右侧常驻显示参会成员与在线状态。
- [x] **移动端抽屉**：小屏自动切换为“参会人(N)”抽屉，不挤占视频主区域。
- [x] **定向私聊入口**：点击参会成员后可发送私聊消息（点对点）。

#### 2. 会议页面与交互优化
- [x] **布局重构**：会议页改为“视频主区 + 参会侧栏”双栏结构。
- [x] **画布信息精简**：视频卡片仅显示用户名 + 麦克风/摄像头状态图标。
- [x] **误触治理**：移除视频点击/双击放大交互，降低误触放大问题。
- [x] **标题栏操作优化**：房间号内联展示，新增“复制会议链接”按钮。

#### 3. 会议技术方案落地
- [x] **跨端入会链路**：支持桌面端与网页端同房间会话（固定远程信令服务）。
- [x] **实时信令同步**：基于轮询+WebRTC SDP/ICE 信令交换，保障多人互联。
- [x] **音视频状态同步**：参会成员麦克风/摄像头状态在画布与侧栏双通道展示。

### v0.8.0 - AI 功能深化 (2026-03-26)

#### 1. 笔记全文问答（RAG）
- [x] **全库问答入口**：在编辑器中新增“笔记问答”图标与弹窗。
- [x] **混合检索链路**：结合 FTS 关键词召回与语义检索（Embedding 可用时）。
- [x] **引用片段返回**：回答附带来源片段，支持点击跳转对应笔记。
- [x] **降级兜底**：Embedding 模型不可用时自动降级到关键词+词面匹配，避免请求直接失败。

#### 2. 多语种翻译增强
- [x] **过期标记策略**：原文更新后不自动覆盖译文，改为“待更新”状态。
- [x] **手动更新入口**：支持按语种单独触发重译。
- [x] **标题一致性**：翻译版本保留原标题，不拼接语种后缀。

#### 3. 编辑体验与界面优化
- [x] **问答弹窗视觉重构**：对齐现有设计系统，优化层级、间距、按钮与卡片样式。
- [x] **右侧快捷操作区**：问答/删除/录音图标布局重构与对齐优化。
- [x] **输入稳定性修复**：修复编辑器重建导致的输入覆盖问题。

### v0.7.0 - AI 智能助手 (2026-03-25)

#### 1. AI 核心能力
- [x] **智能总结**：基于 LangChain 自动提取笔记内容摘要。
- [x] **语音实时润色**：修正语音录入中的语病、错别字，去除口癖。
- [x] **多模型接入**：支持自定义 Base URL，适配所有主流大模型。

#### 2. 总结历史管理
- [x] **独立历史记录**：总结结果保存于笔记元数据，不干扰正文编辑。
- [x] **历史侧边栏**：右侧抽屉展示所有历史生成的摘要。
- [x] **一键引用**：将特定历史总结插入正文。

#### 3. 配置管理
- [x] **AI 参数持久化**：在 UI 中配置 API Key、Base URL、Model Name。
- [x] **交互反馈**：实时显示“润色中”、“生成中”等任务状态。
